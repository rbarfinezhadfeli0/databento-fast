Installation:
  pip install databento-fast

Basic Usage:
  import databento_cpp
  
  records = databento_cpp.parse_file_mbo_fast("data.dbn")
  print(f"Processed {len(records):,} records")
  
  for r in records[:5]:
      print(f"Price: ${r.price_float:.2f}, Size: {r.size}")

With pandas:
  import databento_cpp
  import pandas as pd
  
  records = databento_cpp.parse_file_mbo_fast("data.dbn")
  df = pd.DataFrame([{'price': r.price_float, 'size': r.size} for r in records])
  print(df.describe())
